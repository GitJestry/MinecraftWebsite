Diese Website dient als zentrale Anlaufstelle, um die Arbeiten des Minecraft-Creators vorzustellen und Downloads bereitzustellen.

## Admin-Preview-Check

Um manuell zu prüfen, dass neue Projekte mit Download-Pfad automatisch eine Version im Changelog erhalten:

1. Im Admin-Bereich ein neues Projekt anlegen und einen Download-Pfad setzen.
2. Speichern, damit der Changelog-Dialog erscheint.
3. Prüfen, dass im Changelog-Eintrag die Version **1.0.0** als Standardwert angezeigt wird.
