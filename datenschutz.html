2. Hosting and server logs 2. Hosting und Server‑Logs

The site is served as static files without a database or application backend. Our hosting platform processes only the technically necessary connection data to deliver the pages securely; we do not create or keep personal server logs beyond what the infrastructure needs for short-term fault detection.

Die Seite wird als statische Dateien ohne Datenbank oder Anwendungs-Backend ausgeliefert. Die Hosting-Plattform verarbeitet nur die technisch notwendigen Verbindungsdaten, um die Seiten sicher bereitzustellen; darüber hinaus legen wir keine personenbezogenen Server-Logs an und bewahren sie nicht auf – abgesehen von kurzzeitigen Protokollen, die der Infrastruktur zur Fehlererkennung dienen.

Processed data Verarbeitete Daten

  • IP address (in shortened or pseudonymised form, where possible)
  • Date and time of the request
  • Requested file and URL
  • Amount of data transferred
  • HTTP status code
  • Browser type and version, operating system
  • IP‑Adresse (soweit möglich gekürzt oder pseudonymisiert)
  • Datum und Uhrzeit der Anfrage
  • Abgerufene Datei und URL
  • Übertragene Datenmenge
  • HTTP‑Statuscode
  • Browsertyp und ‑version, Betriebssystem

The legal basis is Article 6(1)(f) GDPR (legitimate interest in the secure and stable operation of the website). Log files are automatically deleted after a short period unless they are needed longer to investigate security incidents.

3. Cookies, local storage & language selection 3. Cookies, Local Storage & Sprachauswahl

This site does not use tracking cookies. We only store a minimal language preference in your browser so that the chosen language persists across page views. No personal profiles are created from this information.

Diese Seite verwendet keine Tracking-Cookies. Es wird lediglich deine Sprachauswahl lokal im Browser gespeichert, damit sie auf allen Seitenaufrufen erhalten bleibt. Aus diesen Informationen werden keine Nutzungsprofile gebildet.

The storage is done using the browser's local storage and technically necessary cookies. This is required so that the website can remember your preferences across page views.

Die Speicherung erfolgt über den Local Storage deines Browsers sowie technisch notwendige Cookies. Dies ist erforderlich, damit sich die Website deine Einstellungen über mehrere Seitenaufrufe hinweg merken kann.

The legal basis is Article 6(1)(f) GDPR (legitimate interest in a user‑friendly presentation of the site).

Rechtsgrundlage ist Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an einer benutzerfreundlichen Darstellung der Website).

Downloads (for example datapacks) are delivered as static files. We do not record download counts, build usage profiles or send download data to third parties. Only the technical request necessary to provide the file is processed by the hosting platform.

Downloads (etwa Datapacks) werden als statische Dateien ausgeliefert. Wir führen keine Download-Zählungen durch, erstellen keine Nutzungsprofile und übermitteln keine Download-Daten an Dritte. Verarbeitet wird nur die technische Anfrage, die zum Ausliefern der Datei benötigt wird.
